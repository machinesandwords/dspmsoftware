DSPM Findings Prioritization Matrix

Platform severity tiers are a starting point, not a remediation sequence. This tool scores each finding across five dimensions — data sensitivity, access exposure, store criticality, finding age, and downstream dependencies — and produces a ranked queue with per-finding rationale structured for handoff to remediation owners.

How to use: Add findings one at a time using the form below. Select the option that best describes current state, not intended or projected state. Once you have your findings in the queue, generate the matrix. Export the ranked output as a CSV when you're ready to assign remediation work.
